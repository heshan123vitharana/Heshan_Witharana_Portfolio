# Heshan Witharana - Modern 3D Portfolio

![Portfolio Preview](public/preview.png)

A high-performance, interactive 3D portfolio website built with modern web technologies. This project showcases advanced frontend development skills, featuring a custom 3D engine substitute (Framer Motion), glassmorphism design, and a fully responsive layout.

## ğŸš€ Features

- **Interactive 3D Hero Section**: A custom-built 3D environment using Framer Motion that responds to mouse movement and drag interactions.
- **Glassmorphism Design**: extensive use of backdrop blur, translucency, and multi-layered glass effects for a premium, futuristic aesthetic.
- **3D Profile Card**: An interactive, multi-layered profile component with depth, parallax, and floating particle effects.
- **Dynamic Backgrounds**: 
  - "Matrix-style" falling code streams with digital overlays.
  - Glowing orbs and geometric shapes floating in 3D space.
- **Smooth Animations**: High-performance animations powered by `framer-motion` for page transitions, scroll reveals, and micro-interactions.
- **Responsive & Mobile-First**: Fully optimized for all devices, from large desktop screens to mobile phones.
- **Project Showcase**: Detailed project cards with hover effects and direct links.
- **Skills & Achievements**: Visual representation of technical expertise and career milestones.

